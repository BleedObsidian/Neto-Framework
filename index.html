Neto-Framework

Neto is an easy to use, lightweight and powerful event driven NIO framework.

Neto gives you the power and tools to easily create server and client connections. Neto supports TCP and UDP with additional RSA encryption functionality. Neto is extremely easy to use allowing you to quickly and efficiently create network applications. The project was initially designed for games but was then adapted for any type of network application. Neto can be used by beginners to experts, this website contains many tutorials on how to use the framework; it also comes with full documentation.

TCP & UDP

Neto allows you to choose what protocols you would like to use for your application. You can switch between protocols easily without having to change all of your code.

Event Driven

Neto is an event drive framework making it very easy and fast to create network applications. The event system allows you to tune in to server and client events of your choice.

RSA Encryption

Neto supports an RSA-Style encryption system to protect and authenticate your network applications.

Flexible

Neto is a very flexible framework; you can create many different types of applications. Neto was initially designed for games but the project was then adapted for general use.
